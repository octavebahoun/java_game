Génération de la présentation PPTX

Prérequis : Python 3 et le module `python-pptx`.

1) Installer la dépendance :

```bash
python3 -m pip install --user python-pptx
```

2) Générer la présentation :

```bash
python3 scripts/generate_presentation.py
```

Le fichier `presentation.pptx` sera créé à la racine du projet.
